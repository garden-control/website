Banco de dados

Banco de dados é um conjunto organizado de informações armazenadas eletronicamente em um sistema de computador. Permite a coleta, armazenamento, recuperação e manipulação de dados de forma eficiente e estruturada. Neste projeto, os sensores enviarão suas informações para um banco de dados que está armazenado na nuvem.
